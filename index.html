purecss-will_paginate for Ruby on Rails

Rails Engine that extends will_paginate stylings to match the pagination styling conventions in Pure CSS toolkit.

Check the gem purecss to include Pure in the Rails Asset Pipeline.

Installation

Add this line to your application's Gemfile:

gem 'purecss-will_paginate'

And then execute:

$ bundle

Or install it yourself as:

$ gem install purecss-will_paginate

Usage

Just like you would in the regular will_paginate. If you've got a need to use the default will_paginate stylings, pass an option like so:

<%= will_paginate(@things, :renderer => WillPaginate::ActionView::LinkRenderer) %>

Based on bootstrap-will_paginate.
